# CUDA Freshman 互动学习日志

本文件按时间追加记录学习会话。每条记录应短而完整，保留可复盘的信息，不保存整段对话。

## 记录模板

```markdown
## YYYY-MM-DD HH:MM CST - session-id

- 章节：chapter_id / chapter_title
- 目标：本轮要解决的一个学习目标
- 预测：用户运行前的判断或疑问
- 命令：实际执行的关键命令
- 观察：关键输出、耗时、错误或现象
- 解释：本轮达成的概念理解
- 改动：代码、参数或环境修改
- 结论：是否完成本轮目标
- 下一步：下一次学习应从哪里继续
```
